Gets the primary term slug from Yoast SEO if available.

tseg_related_content_primary_or_all_slugs(int $post_id, string $taxonomy) : string

If the Yoast SEO plugin is active and a primary term is set for the given post and taxonomy, its slug is returned. Otherwise, it falls back to a comma-separated string of all assigned term slugs.

Parameters
$post_id : int

The ID of the post to check.

$taxonomy : string

The slug of the taxonomy to query (e.g., 'category').

Return values
string —

A single primary term slug or a comma-separated list of all slugs. Returns an empty string if no terms are found.
